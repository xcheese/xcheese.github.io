# 平台管理后台产品原型

面向 Patreon / Ko-fi / Etsy / Gumroad 类平台的官方运营后台高保真前端原型。覆盖工作台、店铺、会员方案、商品、用户、订单、退款与结算、客诉工单、流量与页面、分析报告、标注与备注、系统设置。

## 查看方式

这是一个无构建依赖的静态 SPA，可直接部署到 GitHub Pages。为了避免浏览器对本地文件的限制，建议在本目录启动静态服务器：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173/`。

## 交互范围

- 左侧全部模块可切换，列表行可进入详情。
- 支持全局搜索、模块筛选、排序、日期范围和岗位视图切换。
- 店铺、方案、商品、用户、订单、客诉均有详情与分析标签页。
- 备注可新增并保存到浏览器 `localStorage`。
- 列表可导出为 CSV；订单与工单有模拟操作反馈。
- 桌面、平板与移动端响应式布局。

当前使用集中模拟数据，不连接生产接口，也不执行真实退款、结算或客服动作。
